# Project Rules — t-d

1. All page styling uses **Tailwind CSS** (except things Tailwind cannot handle).
2. Use only **already-installed libraries** for graphs (`recharts`) and icons (`lucide-react`).
3. Add **comments** throughout all code to keep it clean and readable.
4. Always use the project **color palette**:
   - `#0A2947` — Brand navy (primary)
   - `#A8C8E8` — Soft sky blue (accent / highlights)
   - `#FFF2DB` — Warm peach (card borders / inputs)
   - `#FFFAF3` — Warm cream (page backgrounds)
5. The **entire website is in Arabic and RTL** (right-to-left). Never use LTR layout or English UI text.
